Résumé et crédits

Ce projet avait pour but de nous remettre dans le bain sur les notions de PHP et de wordpress apprise la session passée en nous demandant de créer un site de recettes. Le focus était donc principalement sur le php et wordpress plutôt que la stylisation du site web

J'ai codé tout le site en HTML CSS et PHP et intégré les ACF et CPT dans wordpress. Les images sont libres de droit ou générés par IA et les textes ont été générés par IA

Création des différentes recettes

Pour faciliter l'intégration de mon contenu dans le site web, j'ai créé un ACF personalisé qui contient les champs à remplir pour remplir les informations d'une nouvelle recette, ensuite, j'ai utilisé cet ACF dans des CPT de recette pour ajouter le contenu
